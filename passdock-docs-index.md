# PassDock 文档索引

## 先看哪份

如果你不想在一堆文档里来回找，直接按这个顺序看：

### 1. 先看系统使用教程

- [系统使用教程](./passdock-system-usage-guide.md)

适合：

- 第一次接手项目
- 想知道系统到底怎么用
- 想知道后台每个页面是干什么的

### 2. 再看后台系统配置说明

- [后台系统配置说明](./passdock-admin-system-config-guide.md)

适合：

- 想按字段配置支付通道、Provider、Action、履约策略、交付策略
- 想知道系统中心每一项具体怎么填

### 3. 再看部署和总说明

- [README.md](./README.md)
- [apps/web/README.md](./apps/web/README.md)
- [passdock.env.example](./passdock.env.example)

适合：

- 本地启动
- Docker / Compose 部署
- 默认账号和运行说明
